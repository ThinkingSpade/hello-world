oracle.exe

Oracle

Multi-agent reasoning engine

Agentic AI · 2026

A council of debating agents, a skeptic, an optimist, and a fact-checker, each one running on a different frontier model, argue over your data until they reach consensus, then hand back the answer with a confidence score drawn straight from where they disagreed.

STACK

    ↳ How it fitsEach agent is a LangGraph node wired to a different model, Claude, GPT, Gemini, and Grok, so they genuinely disagree; a moderator node weighs the arguments and turns the spread of opinion into the confidence score, so the disagreement is the metric.

    armada.fleet

    Armada

    A control plane for browser fleets

    Distributed Systems · Agentic Automation · 2026

    A control plane for running fleets of real browsers as a managed service. Operators compose multi-tenant campaigns in a React console, a scheduler spreads them across the day, and every run drives a live Chromium session by handing the page's accessibility tree to a frontier model that replies with one schema-checked action.

    STACK

      ↳ How it fitsUnder the hood is the unglamorous platform that keeps it alive: a scheduler with cron, jitter, and ramp-up, warmed persistent identities pulled from a pool, health-scored proxies that disable themselves, classified failures with alerting, and cost metered against a hard ceiling so a runaway run halts itself. Every action is a strict JSON schema, so the model can only click, fill, scroll, or stop, and I can VNC straight into any live worker to watch it think. All of it on Postgres and Redis, running unattended on a single Mac Mini in the corner.

      operator.call

      AI receptionist that answers the phone

      Voice AI · Twilio · 2026

      An AI voice agent that picks up real phone calls, talks to the caller in natural speech, works out what they need, and books them straight onto a live calendar. No hold music, no phone tag. It runs on a real Twilio number, so to the caller it is just someone answering the phone.
